Aplicaciones web, Android y videojuegos

¡Hola! Somos un grupo de programadores apasionados por las nuevas tecnologías. Nos especializamos en desarrollo web responsive y el diseño de videojuegos.

Desarrollo web

Creamos tu página web a la medida de tus necesidades. También podemos reforzar la seguridad y velocidad de tu sitio así como mejorar tu SEO con un mantenimiento continuado.

Proyectos a medida

Tiendas online, aplicaciones de facturación, miniservidores, sistemas empotrados… Gracias a frameworks como Laravel y Django podemos crear una aplicación personalizada que vaya más allá de una web para anunciarse.

Videojuegos

Si siempre has querido hacer un videojuego pero no sabes cómo, cuéntanos. A través de frameworks como libGDX podemos crear juegos multiplataforma que se ejecuten web, móvil y escritorio.
